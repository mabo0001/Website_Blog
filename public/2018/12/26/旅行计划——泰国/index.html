地理位置


曼谷&芭提雅

出行准备及须知

签证

提前1个月，淘宝上可办，或其他方式。一般的，办理时只需提供护照、照片。

出入境规定

  1. 登机前出示护照、签证、全部抵达返回卡。入境航班发还。
  2. 飞行途中填写 出入境登记卡。
  3. 下机后移民局柜台递上护照+出入境登记表、护照盖章、停留时间注明、处境表钉在护照内页，不可遗失。
  4. 出境时收处境卡，办离境手续，护照盖出境章。交机场费500B。出境携带外币不得超过 2K$ or 5WB。
  5. 海关通道：
    • 绿色：随身物品不超过1WB，200根香烟/250g烟草，最大1L酒精类饮料。
    • 红色：有申报物品，交税物品，携带物品总值大于1WB，有违禁物品、受限制物品。
  6. 退税：回国时可要求退还7%增值税，注意提前写好退税单。拿护照向店家拿单子，同一天内同一家百货公司购买超过2KB以上商品、总消费金额达到5KB以上可退税。（详见后一节）
  7. 以下为出入境登记卡具体填写内容：

气候

  • 11月~次年2月 => 凉季，较干燥，约18~26℃。
  • 3月~5月 => 热季，有时40~42℃。
  • 6月~10月 => 雨季，多为短时过云雨，偶有台风，约27~29℃。

衣物

  • 寺庙禁短裤短裙，适当正式。
  • 泳衣泳裤，人字拖，外套开衫（空调很冷）

兑换

提前兑换一些外币，付小费、买小物用。机场、当地尽量不换（十分不划算）。

电源插座

220V/50Hz二孔插座，国内二相插座均可直接使用。三相插座和中国不同，因此中国三相插头需使用二相转换。

注意事项

小费

泰国人大多没有给小费的习惯，但是在中高档餐厅或者酒店享受服务时，可以酌情给予一定小费。20-50铢，或是消费额的10%。行李、床头可20 泰铢，massage 为 50～100 泰铢。

有的餐馆有Service Charge，可不用给小费。付小费时不要给硬币，一般硬币是给乞丐的。

风俗&禁忌

  1. 店内车上禁榴莲
  2. 禁吸烟赌博（包括扑克）
  3. 寺庙、教堂、宫殿、高级娱乐场所为公认的正式场合，进入时，衣着须端庄整洁，不可袒胸露背、穿拖鞋、短裤、迷你裙、无袖上衣或其他不适宜的衣服。
  4. 不能用手指僧侣，不能接触僧侣（身体），尤其女性不能与僧侣握手，在汽车上不能与僧侣邻坐。
  5. 绝对不可爬上佛像拍照，不可对佛像作出有损尊敬的举动；遇见托钵化缘的和尚，不可送现金，因为这是破坏僧侣戒律的行为。
  6. 在进入到佛堂、回教寺或私人住宅时，游客需要脱鞋，并注意不可脚踏门槛。
  7. 不要触及他人头部，也不要弄乱他人的头发。当地人忌讳外人抚摸小孩（尤其是小和尚）的头部，小孩的头只允许国王、僧侣和自己的父母抚摸。
  8. 当地人认为人的左手不洁，因此重要东西要用右手拿；脚掌也被认为是不净的，在入坐时，应避免将脚放在桌子上；用脚尖撞人或指人都会被严厉地呵斥，也绝对不能把脚掌冲向佛。

退税

当天在同一商场购买2000泰铢以上并印有“VAT Refund For Tourists”字样的商品，可以向商家索取退税表格和商品收据。

在国际机场离境时，记得一定要在办理行李托运前向海关(到素万那普机场4楼大厅后找Customs Inspection for VAT Refund的标志)出示购买的物品和P.P.10表，海关盖章后才能托运行李。

注意超过10000铢的小件贵重物品(珠宝，首饰，手表等)不能托运，而要拿到机场退税办理处做第二次检查。 过完边检后带着有海关盖章的P.P.10表和原始购物发票，去退税办理处退税。现金退税手续费100铢，汇票和汇款退税的转账费另算。
素万那普机场的退税办理处(VAT Refund for Tourists Office)在4楼东西两侧各有一个，都能方便找到，电话+662-134-0675-6。

其它机场退税地点及更多信息请参考官网

    • 从素旺那普机场客运站、曼谷市区(胜利纪念碑、北部汽车站、南部汽车站、东部汽车站)都有前往芭堤雅的面包车，用时约2个小时，车费：200泰铢，运行时间为早上5点至晚上10点，具体运行时间可致电各汽车站。
  1. 大巴：
    • 从素旺那普机场的客运站也有公车前往芭堤雅，票价约100泰铢。
    • 现在开通了从Mochit北部汽车站，Ekkamai东部汽车站以及Sai Dai Mai南部汽车站分别开往芭堤雅的空调巴士。

    • 北部汽车站的班次也是每30分钟一班，运行时间4:00-22:00，票价：133泰铢；
    • 南部汽车的班车每一小时一趟，运行时间为 5:00-19:00，票价：124泰铢。
    • 全程用时约2个小时。
    • 如果是从廊曼机场前往芭堤雅，可先前往Mo chit汽车站，然后转乘巴士前往芭堤雅。
  • 东部客运站电话：02-391-2504；02-391-8097
  • 北部客运站电话：02-936-2852至66；02-936-3666
  • 南部客运站电话：02-894-6122

住宿位置

Wave Hotel Pattaya / Baraquda Pattaya - MGallery by Sofitel / AVANI Pattaya Resort

格兰岛

  • 出行：从芭提雅海滩快艇（约200B私人）or轮渡（30B交通工具）前往。

  • 轮渡发船时间：7:00、10:00、12:00、14:00、15:30、17:00、18:30供参考。人满即走。约40min行程。【返程末班注意为18:00！！！】

  • 萨美海滩，欧美人较多；其他国人很多，环境略差。（游客最为常去的沙滩是Tawaen，其次是Tian、Tonglang、Samae）

  • 娱乐活动：
    • 拖曳伞，点水时跑起来不然容易伤脚，有一定危险性。注意！
    • 租车逛时不要摩托。拿车前无死角拍照以免被坑，停车记得锁。仍不建议租车。很多大拐弯和大角度的上下坡。
    • 海底漫步、浮潜时注意不要人多时参与，水浑浊，可切换海滩or错开时间。
    • 摩托艇塞小费可自驾玩。
    • 考虑淘宝的岛上一日游？
    • 大法万岁，妥妥防水。
    • 随时砍价。。。随时问清楚货币单位。


萨美海滩：经芭提雅港乘轮渡，时间无参考意义。

真理寺

  • 出行：海滩附近乘双条车200B/2人。（参考）

  • 票价：450B，平台提前买可便宜。

  • 真理圣殿建于1981年，选址为芭提雅一片临海的宁静之地。至今已有31年的历史，还没有竣工，建筑是全木纯手工雕刻，构造结构采用的是卡榫结构，非常的年轻，但是从外边看去的时候好像非常的古老，因为整个建筑采用的木头都是至少年龄为150年的古木，所以欣赏的时候会散发出一种古气。由于建筑建于海边，每天都会受到海风的侵蚀，所以所有的雕刻在完成以后都会涂上一种保护油（保护油是无色的），时间久了会产生一种像青苔一样的绿色，使整个建筑看起来更加的古老。建筑的外面或者里面林立着有许多佛像或者仙女，一些游客会认为这里是一座寺庙，那是不对的。它只是一个艺术品，创始人创建的目的只是为了保留泰国的木质工艺和雕刻艺术。真理圣殿与其它富丽堂皇的寺庙景点不一样，它是艺术品，也是各宗教文化与各哲学的结合体。它的底部是一个十字架，长度分别各为100米，高度为105米。它是整个东南亚宗教文化的融合，设计风格也是多个国家的，有印度、柬埔寨、缅甸、泰国、老挝、中国等等。

  • 游客来到这里除了可以欣赏到世上独一无二的艺术还可以参与休闲活动，游客在欣赏艺术的同时还可以骑大象、骑马、骑马车、坐木船、做游艇、欣赏泰国传统舞蹈和剑术表演，还可以给小鲨鱼喂食。
